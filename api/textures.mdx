## Overview

Vista's texture system handles dynamic textures for cassette tapes, web content, and live camera feeds. It includes support for GIF animations, sprite strips, and real-time rendered textures.

## LiveFeedTexture

`LiveFeedTexture` (`net.mehvahdjukaar.vista.client.textures.LiveFeedTexture`) manages render-to-texture for viewfinder camera feeds.

### Constructor

```java

    UUID id,
    @Nullable ResourceLocation postFragment
)
```

<ParamField path="resourceLocation" type="ResourceLocation" required>
  The resource location identifier for this texture.
</ParamField>

<ParamField path="size" type="int" required>

</ParamField>

<ParamField path="textureDrawingFunction" type="Consumer<LiveFeedTexture>" required>

### Loading Process

<CodeGroup>
```java Load Texture
@Override
public void load(ResourceManager resourceManager) throws IOException {
    SpriteContents spriteContents = loadContent(resourceManager);
    if (spriteContents == null) return;
    
    this.stripData = AnimationStripData.create(spriteContents);
    
    NativeImage nativeImage = spriteContents.originalImage;
    if (!RenderSystem.isOnRenderThreadOrInit()) {
        RenderSystem.recordRenderCall(() -> this.doLoad(nativeImage));
    } else {
        this.doLoad(nativeImage);
    }
}
```

```java Content Loading
private SpriteContents loadContent(ResourceManager rm) throws FileNotFoundException {
    Resource resource = rm.getResourceOrThrow(this.fileLocation);
    SpriteResourceLoader loader = fileLocation.getPath().endsWith(".gif") ?
        GifPathSpriteSource.GIF_CONTENT_LOADER : RESHAPING_PNG_STRIP_LOADER;
    return loader.loadSprite(fileLocation, resource);
}
```
</CodeGroup>

### Texture Reshaping

For large animations exceeding GPU texture limits:

```java Atlas Layout
private static NativeImage buildTileAtlasFromStrip(
    NativeImage originalImage,
    int frameW,
    int frameH,
    int frameCount,
    int maxTextureSize
) {
    // Compute optimal layout
    Vec2i layout = computeAtlasLayout(
        frameCount, frameW, frameH,
        maxTextureSize, maxTextureSize
    );
    int rows = layout.x();
    int cols = layout.y();
    
    int atlasW = cols * frameW;
    int atlasH = rows * frameH;
    
    NativeImage out = new NativeImage(
        NativeImage.Format.RGBA, atlasW, atlasH, true
    );
    
    // Copy frames in column-major order
    for (int i = 0; i < frameCount; i++) {
        int row = i % rows;
        int col = i / rows;
        
        int xOff = col * frameW;
        int yOff = row * frameH;
        
        originalImage.copyRect(
            out, 0, i * frameH,
            xOff, yOff, frameW, frameH,
            false, false
        );
    }
    
    return out;
}
```

### Async Loading

```java
@Override
public void load(ResourceManager resourceManager) throws IOException {
    if (this.future == null) {
        NativeImage nativeImage;
        
        // Try loading from cache first
        if (this.cachelLocation != null && this.cachelLocation.isFile()) {
            VistaMod.LOGGER.debug("Loading http texture from local cache ({})",
                this.cachelLocation);
            FileInputStream fis = new FileInputStream(this.cachelLocation);
            nativeImage = readImageFromStream(fis);
        } else {
            nativeImage = null;
        }
        
        if (nativeImage != null) {
            this.onLoaded(nativeImage);
        } else {
            // Download asynchronously
            this.future = CompletableFuture.runAsync(() -> {
                HttpURLConnection conn = null;
                try {
                    conn = (HttpURLConnection) (new URL(this.urlString))
                        .openConnection(Minecraft.getInstance().getProxy());
                    conn.setDoInput(true);
                    conn.setDoOutput(false);
                    conn.connect();
                    
                    if (conn.getResponseCode() / 100 == 2) {
                        // Success - cache and load
                        if (this.cachelLocation != null) {
                            FileUtils.copyInputStreamToFile(
                                conn.getInputStream(), this.cachelLocation
                            );
                        }
                        NativeImage ni = readImageFromStream(inputStream);
                        if (ni != null) {
                            this.onLoaded(ni);
                        }
                    }
                } catch (Exception e) {
                    VistaMod.LOGGER.error("Couldn't download http texture", e);
                } finally {
                    if (conn != null) conn.disconnect();
                }
            }, Util.backgroundExecutor());
        }
    }
}
```

## GIF Support

### GifPathSpriteSource

Handles GIF file loading and conversion to sprite strips:

```java
public class GifPathSpriteSource {
    public static final SpriteResourceLoader GIF_CONTENT_LOADER = 
        (resourceLocation, resource) -> {
            // Load GIF frames
            List<NativeImage> frames = loadGifFrames(resource);
            
            // Determine frame size
            int frameW = frames.get(0).getWidth();
            int frameH = frames.get(0).getHeight();
            
            // Create vertical strip or tiled atlas
            NativeImage strip = createStripFromFrames(frames, frameW, frameH);
            
            // Create sprite contents with animation metadata
            return new SpriteContents(
                resourceLocation,
                new FrameSize(frameW, frameH),
                strip,
                buildAnimationMetadata(frames)
            );
        };
}
```

## Custom Vertex Consumers

### AnimatedStripVertexConsumer

Samples from animated sprite strips:

```java
public class AnimatedStripVertexConsumer implements VertexConsumer {
    private final VertexConsumer delegate;
    private final AnimationStripData stripData;
    private final int currentFrame;
    
    public AnimatedStripVertexConsumer(
        VertexConsumer delegate,
        AnimationStripData stripData,
        int currentFrame
    );
    
    @Override
    public VertexConsumer uv(float u, float v) {
        // Remap UVs to current frame
        float frameU = stripData.getFrameU(currentFrame);
        float frameV = stripData.getFrameV(currentFrame);
        float frameW = stripData.getFrameWidth();
        float frameH = stripData.getFrameHeight();
        
        float remappedU = frameU + u * frameW;
        float remappedV = frameV + v * frameH;
        
        return delegate.uv(remappedU, remappedV);
    }
}
```

## Example Usage

### Creating a Live Feed Texture

<CodeGroup>
```java Basic Live Feed
UUID feedId = viewfinder.getUUID();
int resolution = 512; // 512x512 pixels

LiveFeedTexture texture = new LiveFeedTexture(
    VistaMod.res("livefeed/" + feedId),
    resolution,
    (tex) -> VistaLevelRenderer.render(tex, viewfinder),
    feedId,
    null // No post-processing
);
```

```java With Post-Processing
LiveFeedTexture texture = new LiveFeedTexture(
    VistaMod.res("livefeed/" + feedId),
    resolution,
    (tex) -> VistaLevelRenderer.render(tex, viewfinder),
    feedId,
    VistaMod.res("shaders/post/ntsc_tv.json")
);
```
</CodeGroup>

### Loading Animated Textures

```java
// Load GIF or animation strip
AnimatedStripTexture texture = new AnimatedStripTexture(
    ResourceLocation.parse("mymod:textures/cassette_tape/my_animation.gif")
);

// Register with Minecraft's texture manager
Minecraft.getInstance().getTextureManager().register(
    texture.getTextureLocation(),
    texture
);

// Get animation data
AnimationStripData data = texture.getStripData();
int frameCount = data.getFrameCount();
int currentFrame = (int) (gameTime % frameCount);
```

### Downloading Web Textures
