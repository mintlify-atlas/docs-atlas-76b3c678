## Detection System

Vista automatically detects installed mods at runtime using the `CompatHandler` class:

```java
public static final boolean EXPOSURE = PlatHelper.isModLoaded("exposure");
public static final boolean DISTANT_HORIZONS = PlatHelper.isModLoaded("distanthorizons");
public static final boolean COMPUTER_CRAFT = PlatHelper.isModLoaded("computercraft");
public static final boolean SUPPLEMENTARIES = PlatHelper.isModLoaded("supplementaries");
public static final boolean IRIS = PlatHelper.isModLoaded("iris") || PlatHelper.isModLoaded("oculus");
public static final boolean SODIUM = PlatHelper.isModLoaded("sodium") || PlatHelper.isModLoaded("embeddium");
```

<Note>
All integrations are optional and will only activate when the corresponding mod is installed. Vista works perfectly fine without any of these mods.
</Note>

## Supported Platforms

Most integrations work on both **Fabric** and **NeoForge** mod loaders, with platform-specific implementations where necessary:

- **ComputerCraft**: Full support on both platforms with peripheral APIs
- **Exposure**: Cross-platform photograph display system
- **Iris/Oculus**: Shader compatibility on both loaders
- **Sodium/Embeddium**: Performance mod variants for different platforms

<Info>
For detailed information about each integration, click on the cards above or navigate to the specific integration page.
</Info>

## Configuration

Some integrations provide configurable options:

- **Distant Horizons**: LOD quality settings for camera feeds
- **Iris**: Toggle shader disabling in Vista screens

These options can be found in Vista's configuration file after the respective mods are loaded.
