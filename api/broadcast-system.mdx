## Overview

The Vista broadcasting system enables live video feeds from ViewFinders to be displayed on TVs across the world. The system is built on a centralized manager that tracks broadcast sources and their subscribers.

### Key Components

- **BroadcastManager**: World-saved data tracking all active broadcasts
- **IBroadcastLocation**: Interface for broadcast source locations
- **IBroadcastSource**: Interface for entities that provide video content
- **BroadcastLocationType**: Type registry for custom location implementations

## BroadcastManager

The central manager for all broadcast feeds in a world dimension.

### Class Definition

```java

```

**Location:** `net.mehvahdjukaar.vista.common.broadcast.BroadcastManager`

### Getting an Instance

```java

```

Retrieves the BroadcastManager for a given world dimension.

<ParamField path="level" type="Level">
  The world level (dimension). Each dimension has its own BroadcastManager instance.
</ParamField>

<ResponseField name="Returns" type="BroadcastManager">

**Example:**

```java

public void manageBroadcasts(ServerLevel level) {

```

### Write Methods (Mutating Operations)

<Warning>
  All write methods automatically mark the manager as dirty and sync to clients. These should only be called on the server side.
</Warning>

#### linkFeed

```java
public void linkFeed(UUID feedUUID, IBroadcastLocation projectorPos)
```

Registers or updates a broadcast feed at a specific location.

<ParamField path="feedUUID" type="UUID">
  The unique identifier for the broadcast feed. Typically the ViewFinder's UUID.
</ParamField>

<ParamField path="projectorPos" type="IBroadcastLocation">
  The location where the broadcast originates (e.g., ViewFinder position).
</ParamField>

**Behavior:**
- If the UUID already exists with a different location, updates the location
- If the UUID is new, creates a new broadcast registration
- Automatically syncs to clients and marks data as dirty

**Example:**

```java
import net.mehvahdjukaar.vista.common.broadcast.LevelBELocation;
import java.util.UUID;

public void registerBroadcast(ServerLevel level, BlockPos pos, UUID feedId) {
    BroadcastManager manager = BroadcastManager.getInstance(level);
    IBroadcastLocation location = LevelBELocation.of(level, pos);
    manager.linkFeed(feedId, location);
}
```

#### unlinkFeed (by Location)

```java
public void unlinkFeed(IBroadcastLocation projectorPos)
```

Removes a broadcast feed at a specific location.

<ParamField path="projectorPos" type="IBroadcastLocation">
  The location to unlink.
</ParamField>

**Use Case:** Called when a ViewFinder block is destroyed.

**Example:**

```java
public void onViewFinderRemoved(ServerLevel level, BlockPos pos) {
    BroadcastManager manager = BroadcastManager.getInstance(level);
    manager.unlinkFeed(LevelBELocation.of(level, pos));
}
```

#### unlinkFeed (by UUID)

```java
public void unlinkFeed(UUID feedUUID)
```

Removes a broadcast feed by its UUID.

<ParamField path="feedUUID" type="UUID">
  The feed UUID to remove.
</ParamField>

**Use Case:** Called when cleaning up orphaned broadcast references.

### Read Methods (Fast, Thread-Safe)

<Note>
  All read methods operate on an immutable snapshot and are thread-safe. They can be called from both client and server sides.
</Note>

#### getBroadcast

```java
public IBroadcastSource getBroadcast(UUID feedId, boolean clientSide)
```

Retrieves the actual broadcast source (e.g., ViewFinder block entity) for a feed.

<ParamField path="feedId" type="UUID">
  The broadcast feed UUID.
</ParamField>

<ParamField path="clientSide" type="boolean">
  True if called from client side, false if server side.
</ParamField>

<ResponseField name="Returns" type="IBroadcastSource">
  The broadcast source, or null if:
  - Feed ID not found
  - Source location is not loaded
  - Block entity at location is not a valid broadcast source
  - Source has been removed (server-side only: automatically unlinks)
</ResponseField>

**Example:**

```java
public void displayFeed(Level level, UUID feedId) {
    BroadcastManager manager = BroadcastManager.getInstance(level);
    IBroadcastSource source = manager.getBroadcast(feedId, level.isClientSide);
    
    if (source != null) {
        IVideoSource video = source.getBroadcastVideo();
        // Render the video...
    }
}
```

#### getAll

```java
public Iterable<Map.Entry<UUID, IBroadcastLocation>> getAll()
```

Returns all registered broadcasts.

<ResponseField name="Returns" type="Iterable<Map.Entry<UUID, IBroadcastLocation>>">
  An iterable of feed UUID to location mappings. Safe to iterate; operates on immutable snapshot.
</ResponseField>

**Example:**

```java
public void listAllBroadcasts(Level level) {
    BroadcastManager manager = BroadcastManager.getInstance(level);
    
    System.out.println("Active broadcasts:");
    for (var entry : manager.getAll()) {
        UUID feedId = entry.getKey();
        IBroadcastLocation location = entry.getValue();
        System.out.println("- Feed " + feedId + " at " + location);
    }
}
```

### Data Persistence

The BroadcastManager uses Minecraft's WorldSavedData system:

```java
public static final Codec<BroadcastManager> CODEC
public static final StreamCodec<RegistryFriendlyByteBuf, BroadcastManager> STREAM_CODEC
```

- **CODEC**: For disk serialization (world save files)
- **STREAM_CODEC**: For network synchronization (client sync)

<Expandable title="Serialization Details">
  The manager serializes as a map of UUID to IBroadcastLocation:
  
  ```java
  Codec.unboundedMap(UUIDUtil.STRING_CODEC, IBroadcastLocation.CODEC)
  ```
  
  When loading:
  1. Deserializes the map
  2. Populates internal tracking structures
  3. Publishes an immutable snapshot for thread-safe reads
  
  The system ensures consistency between server and client by syncing whenever broadcasts change.
</Expandable>

### Implementation Details

#### get Method

```java
@Override
public TriResult<IBroadcastSource> get(boolean isClient)
```

**Behavior:**

1. Resolves the world dimension:
   - Client: Uses local world cache
   - Server: Uses server's dimension registry

2. Checks if the chunk is loaded

3. Retrieves the block entity at the position

4. Returns:
   - **Valid**: If block entity implements IBroadcastSource
   - **Invalid**: If block entity exists but is not a broadcast source (server-side only)
   - **Empty**: If dimension/chunk not loaded

<Note>
  On the server side, returning Invalid signals that the broadcast should be unlinked (e.g., block was replaced).
</Note>

#### getTooltipComponent Method

```java
@Override
public MutableComponent getTooltipComponent(Level level)
```

**Returns:**
- Same dimension: "Linked to X: 100, Y: 64, Z: 200"
- Different dimension: "Linked to minecraft:the_nether"

### Implementation Example

```java
import net.mehvahdjukaar.vista.common.cassette.IBroadcastSource;
import net.mehvahdjukaar.vista.common.broadcast.LevelBELocation;
import net.mehvahdjukaar.vista.client.video_source.IVideoSource;
import java.util.UUID;

public class CustomCameraBlockEntity extends BlockEntity implements IBroadcastSource {
    
    private UUID broadcastId;
    private IVideoSource videoSource;
    
    public CustomCameraBlockEntity(BlockPos pos, BlockState state) {
        super(CUSTOM_CAMERA_TYPE.get(), pos, state);
        this.broadcastId = UUID.randomUUID();
        this.videoSource = createVideoSource();
    }
    
    @Override
    public UUID getUUID() {
        return broadcastId;
    }
    
    @Override
    public IVideoSource getBroadcastVideo() {
        return videoSource;
    }
    
    @Override
    public void setLevel(Level level) {
        super.setLevel(level);
        // Register with broadcast manager
        this.ensureLinked(level, LevelBELocation.of(this));
    }
    
    @Override
    public void setRemoved() {
        super.setRemoved();
        // Unregister from broadcast manager
        if (level != null) {
            this.removeLink(level);
        }
    }
    
    @Override
    protected void saveAdditional(CompoundTag tag, HolderLookup.Provider registries) {
        super.saveAdditional(tag, registries);
        tag.putUUID("BroadcastId", broadcastId);
    }
    
    @Override
    protected void loadAdditional(CompoundTag tag, HolderLookup.Provider registries) {
        super.loadAdditional(tag, registries);
        this.broadcastId = tag.getUUID("BroadcastId");
    }
}
```

## TriResult

A tri-state result type used for broadcast source lookups.

### States

<ParamField path="Valid" type="TriResult<T>">
  Operation succeeded, value is present and valid.
</ParamField>

<ParamField path="Invalid" type="TriResult<T>">
  Operation encountered an invalid state (e.g., wrong block entity type).
</ParamField>

<ParamField path="Empty" type="TriResult<T>">
  Operation could not complete due to temporary conditions (e.g., chunk not loaded).
</ParamField>

### Methods

```java
boolean isValid()
T getValue()
```

### Usage Pattern

```java
TriResult<IBroadcastSource> result = location.get(isClient);

if (result.isValid()) {
    IBroadcastSource source = result.getValue();
    // Use the source
} else if (!isClient && result == TriResult.invalid()) {
    // Server-side cleanup: unlink invalid broadcasts
    manager.unlinkFeed(feedId);
}
// Empty state: do nothing, chunk may not be loaded
```

## Best Practices

<Warning>
  **Memory Leak Prevention**: Always unlink broadcasts when removing block entities. Failure to do so will cause the BroadcastManager to retain stale references.
</Warning>

<Tip>
  **Thread Safety**: Read methods on BroadcastManager are thread-safe and can be called from rendering threads. Write methods should only be called on the server thread.
</Tip>

<Note>
  **Client-Server Sync**: The BroadcastManager automatically syncs to clients when broadcasts change. Client-side read operations work on the synced snapshot.
</Note>

### Common Patterns

#### Registering a Broadcast on Load

```java
@Override
public void setLevel(Level level) {
    super.setLevel(level);
    if (level instanceof ServerLevel) {
        this.ensureLinked(level, LevelBELocation.of(this));
    }
}
```

#### Cleaning Up on Remove

```java
@Override
public void setRemoved() {
    super.setRemoved();
    if (level instanceof ServerLevel sl) {
        BroadcastManager.getInstance(sl).unlinkFeed(LevelBELocation.of(this));
    }
}
```

#### Subscribing to a Feed

```java
public void subscribeToBroadcast(Level level, UUID feedId) {
    BroadcastManager manager = BroadcastManager.getInstance(level);
    IBroadcastSource source = manager.getBroadcast(feedId, level.isClientSide);
    
    if (source != null) {
        IVideoSource video = source.getBroadcastVideo();
        if (video != null) {
            // Display the video
        }
    }
}
```
